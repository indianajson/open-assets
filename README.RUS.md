<h2>Известные ограничения</h2>
- <b>Предпросмотр фонов</b> неточно отражает скорость движения фона, поскольку в игре он будет отображаться с теми же значениями скорости. Для более точного отображения движения фона потребуется переработка рендерера.<br>
- <b>Предпросмотр тайлов/объектов</b> иногда показывает анимацию для другого тайла при слишком быстрой прокрутке. Эта проблема решается перемещением на одну строку вниз или вверх и обратно.
